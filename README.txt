
nobt — Static e-commerce site (estático)
=======================================

Arquivos incluídos:
- index.html  -> Página principal
- styles.css  -> Estilos (preto + dourado)
- script.js   -> Lógica do carrinho + link do WhatsApp (checkout externo)
- products.json -> Lista de produtos
- assets/placeholder.png -> Imagem placeholder (você pode substituir pelos seus arquivos)

Instruções rápidas:
1) Extraia o zip em uma pasta.
2) Abra index.html no navegador para ver localmente.
3) IMPORTANT: abra script.js e substitua a string '55NUMERODOSEUWHATS' pelo seu número completo do WhatsApp (ex: 5511999999999).
4) Para publicar: faça upload dos arquivos para qualquer hospedagem estática (Netlify, Vercel, GitHub Pages, Hostinger, etc).
5) Para imagens de produtos, coloque arquivos em /assets nomeados como: nobt-ice-mint.png, nobt-blue-razz.png, etc.
   Se não houver imagem, o placeholder será mostrado automaticamente.
